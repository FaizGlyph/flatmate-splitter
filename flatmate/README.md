# Flatmate Expense Splitter 💸

A web app to track and split expenses between flatmates.

## 🚀 Features
- Add flatmates
- Track expenses
- Calculate balances
- See who owes whom
- Clean dashboard UI

## 🛠 Tech Stack
- Java (Spring Boot)
- HTML, CSS (Thymeleaf)
- Maven

## 🌐 How to Run
1. Clone repo
2. Run:
   mvn spring-boot:run
3. Open:
   http://localhost:8081
